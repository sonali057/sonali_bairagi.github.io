Karnataka Dashboards

Interactive Guide for Tap Water Connection Dashboard :
  • In the top right corner, there are two dropdown box where users can choose a specific district and then further refine the data by selecting a particular block within that district.
  • Hover the cursor over the map to see the details of the district with tap connection
  • The bar chart displays a comparison of the number of households with functional tap water connections across different blocks.
    • 
    
     

        * Coverage= (Number of households with Tap Connection/Total no. of Households)*100
        
    
    
    
     
    
    
    
    Interactive Guide for LPCD Dashboard:
    
  • In the dropdown box the user can choose a specific district.
    • 
    
  • The bar chart visually represents the comparison of LPCD across different districts.
    • 
    
  • The card displays the average LPCD (Litres Per Capita per Day) value calculated across all districts or
        based on the selection made in the slicer.

    Interactive Guide for Water Quality Dashboard:
    
  • The first dropdown box enables users to select and filter data based on district names within the specified region, 
        while the second dropdown allows users to filter data for a specific year
    • 
    
  • The line chart helps visualize how chemical parameters vary across districts over the selected year. Hover the cursor over the points 
        where it displays the average values of chemical parameters.
    • 
    
  • The bar graph allows users to compare the levels of physical parameters among districts.
    • 
    
  • The upper left bar graph shows the number of samples tested by different labs and the constant line in the graph represents the benchmark which is 3000 samples
    • 
    
  • The user can move to second page of the dashboard from the arrow given in the bottom of the dashboard display. This
        dashboard represents the water potability across all the districts of the selected year.
